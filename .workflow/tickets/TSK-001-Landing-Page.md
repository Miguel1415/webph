# Ticket: TSK-001 - Photographic Landing Page

## Descripción
Crear una interfaz de inicio (landing page) profesional para una web de fotografía. La página debe destacar el portafolio del fotógrafo y permitir la venta de fotografías de eventos y cuadros.

## Criterios de Aceptación
1. **Diseño Premium:** Estética fotográfica, uso de glassmorphism, tipografía moderna y paleta de colores armónica (modo oscuro sugerido).
2. **Secciones Esenciales:**
   - Hero con fotografía de alto impacto.
   - Galería destacada (eventos y cuadros).
   - Sección sobre el fotógrafo.
   - Llamado a la acción (CTA) para compra/contacto.
3. **Interactividad:** Micro-animaciones al hacer scroll y hover en imágenes.
4. **Responsividad:** Totalmente adaptable a dispositivos móviles y escritorio.
5. **Arquitectura:** Estructura de archivos limpia y modular.

## Roles Involucrados
- **Ayudante Jefe de Producto:** Definición de requisitos.
- **Diseñador UX/UI:** Concepto visual y estética.
- **Arquitecto de Software:** Estructura de componentes.
- **Desarrollador Senior:** Implementación técnica.
