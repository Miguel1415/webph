# Ticket: TSK-001 - Landing Page para AuraPH

## Descripción
Crear una interfaz de inicio (landing page) profesional y atractiva para el portfolio de fotografía AuraPH. La página debe destacar el trabajo del fotógrafo, integrar visualmente el dominio `www.auraph.cl` y permitir una fácil navegación hacia el resto del portafolio, funcionando como el punto de entrada principal para clientes y visitantes.

## Criterios de Aceptación
1. **Diseño Premium:** Estética fotográfica, uso de glassmorphism, tipografía moderna y paleta de colores armónica (modo oscuro sugerido).
2. **Secciones Esenciales:**
   - Hero con fotografía de alto impacto y llamado a la acción.
   - Resumen del portafolio (ej. un slider de trabajos recientes).
   - Sección "Sobre Mí" con una breve biografía e imagen del fotógrafo.
   - Testimonios de clientes satisfechos.
   - Contacto (formulario).
3. **Interactividad:** Micro-animaciones sutiles al hacer scroll y hover en elementos clave para mejorar la experiencia de usuario.
4. **Responsividad:** Totalmente adaptable a dispositivos móviles, tablets y escritorio.
5. **Arquitectura:** Estructura de archivos limpia y modular, utilizando HTML/CSS/JS (con Vite) para el frontend, diseñado para un despliegue óptimo en Vercel.

## Roles Involucrados
- **Ayudante Jefe de Producto:** Definición y priorización de requisitos para la landing page de AuraPH.
- **Diseñador UX/UI:** Concepto visual, estética y experiencia de usuario.
- **Arquitecto de Software:** Definición de la estructura de componentes frontend y consideración de la integración con el backend Node.js (si aplica para funcionalidades futuras en la landing) y Supabase.
- **Desarrollador Senior:** Implementación técnica del frontend de la landing page.
- **Ingeniero QA:** Pruebas de funcionalidad y responsividad de la landing page.
