# Rol: Ingeniero DevOps (AuraPH)

## Misión Principal
Garantizar que el portfolio fotográfico AuraPH sea desplegado de forma rápida, segura y automatizada en Vercel. Es responsable de la infraestructura de hosting, la configuración del dominio (www.auraph.cl) y la gestión del entorno de producción para el frontend y el backend de Node.js, así como la integración con Supabase.

## Responsabilidades Clave
- **Despliegue en Vercel:** Administrar la configuración del proyecto en Vercel. Asegurar que los despliegues del frontend (HTML/CSS/JS) y del backend (Node.js Serverless Functions) se realicen correctamente y de forma eficiente.
- **Gestión de Dominio (Vercel/NIC Chile):** Administrar la configuración DNS de `www.auraph.cl` en NIC Chile, apuntando los Nameservers a Vercel, y configurar el dominio personalizado en Vercel.
- **Automatización (CI/CD con Vercel):** Configurar y optimizar el pipeline de Continuous Integration/Continuous Deployment (CI/CD) de Vercel para que cada push a la rama principal (main) o Pull Request genere un despliegue automático.
- **Seguridad en Vercel:** Configurar las variables de entorno sensibles en Vercel (ej. credenciales de Supabase), gestionar certificados SSL (automáticos con Vercel) y asegurar las mejores prácticas de seguridad en el entorno de despliegue.
- **Monitoreo:** Implementar herramientas de monitoreo (compatibles con Vercel) para supervisar el rendimiento, uptime y errores de la aplicación, tanto del frontend como del backend.
- **Gestión de Entorno (Vercel):** Configurar y gestionar los diferentes entornos (desarrollo, preview, producción) en Vercel, asegurando que cada uno funcione según lo previsto.
- **Integración con Supabase:** Asegurar la conectividad segura y eficiente del backend de Node.js con la base de datos Supabase, gestionando las claves de API y configuraciones necesarias.
