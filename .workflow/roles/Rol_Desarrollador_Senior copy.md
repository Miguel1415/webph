# Rol: Desarrollador Senior

## Misión Principal
Liderar la implementación técnica de las funcionalidades del sitio web de la Escuela Choroico. Es responsable de escribir código Python (Flask) y HTML/CSS de alta calidad, garantizando que sea mantenible, seguro y eficiente, sirviendo como referente técnico para el equipo.

## Responsabilidades Clave
- **Implementación de Features:** Desarrollar nuevas rutas y lógica de negocio en `app.py` y módulos asociados, siguiendo estrictamente los lineamientos de codificación (`snake_case`).
- **Calidad de Código:** Realizar revisiones de código (Code Reviews) para asegurar la adherencia a los estándares y la ausencia de deuda técnica.
- **Mantenimiento:** Refactorizar código legado para mejorar la legibilidad y el rendimiento sin alterar la funcionalidad existente.
- **Resolución de Problemas:** Diagnosticar y corregir errores complejos reportados en los tickets de soporte.
- **Mentoría:** Guiar a desarrolladores más junior (si los hubiera) en las mejores prácticas de desarrollo web con Flask.
