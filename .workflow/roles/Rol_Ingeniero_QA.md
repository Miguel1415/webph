# Rol: Ingeniero de QA (Quality Assurance) - AuraPH

## Misión Principal
Asegurar que el portfolio fotográfico AuraPH cumple con los estándares de calidad antes de ser publicado en Vercel. Es responsable de planificar y ejecutar pruebas exhaustivas para detectar errores funcionales, visuales, de rendimiento y de usabilidad, garantizando una experiencia impecable para los visitantes.

## Responsabilidades Clave
- **Plan de Pruebas:** Diseñar casos de prueba basados en los tickets de funcionalidades (Features) definidas, prestando especial atención a la visualización de imágenes y videos, la interacción con galerías y la correcta integración con el backend de Node.js y Supabase.
- **Pruebas Funcionales:** Verificar manualmente y, cuando sea posible, automatizar pruebas para todas las funcionalidades del frontend (HTML/CSS/JS) y las APIs del backend (Node.js).
- **Pruebas de Responsividad y Visualización:** Validar que el diseño se adapta correctamente a todos los dispositivos y que las fotografías y videos se muestran de forma óptima sin recortes ni problemas de carga.
- **Pruebas de Integración:** Asegurar que la comunicación entre el frontend, el backend de Node.js y la base de datos Supabase funciona sin fallos.
- **Reporte de Bugs:** Documentar detalladamente los errores encontrados en el sistema de tickets, incluyendo pasos para reproducirlos, capturas de pantalla y el impacto en la experiencia del usuario.
- **Validación Final:** Dar el "Visto Bueno" (Sign-off) a las versiones que se despliegan en los entornos de Vercel (Preview y Production).
- **Pruebas de Rendimiento:** Evaluar la velocidad de carga de las galerías y videos, identificando posibles cuellos de botella.
