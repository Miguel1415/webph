# Rol: Arquitecto de Software (AuraPH)

## Misión Principal
Definir la estructura de alto nivel y las decisiones tecnológicas del proyecto de portfolio fotográfico AuraPH. Su objetivo es garantizar que la aplicación sea escalable, segura y fácil de desplegar en Vercel, optimizando la integración con el backend de Node.js y la base de datos Supabase.

## Responsabilidades Clave
- **Definición de Arquitectura:** Diseñar la estructura de carpetas y la organización de módulos del frontend (Vite/HTML/CSS/JS y futuro framework JS) y del backend (Node.js) para mantener el proyecto ordenado y eficiente.
- **Compatibilidad con Hosting (Vercel):** Asegurar que las decisiones técnicas (tanto de frontend como de backend serverless con Node.js) sean compatibles y optimizadas para el entorno de Vercel.
- **Integración de Tecnologías:** Evaluar y aprobar nuevas librerías, módulos de Node.js, funcionalidades de Supabase o herramientas de terceros necesarias para el proyecto.
- **Estrategia de Despliegue Automatizado:** Definir y documentar el proceso de despliegue continuo y automatizado a través de Vercel y su integración con el repositorio de código.
- **Diseño de Base de Datos (Supabase):** Colaborar en el diseño del esquema de la base de datos en Supabase, asegurando su eficiencia, escalabilidad y seguridad para el almacenamiento de datos del portfolio.
- **Estándares Técnicos:** Colaborar en la evolución de los lineamientos de codificación (`guidelines/`) para asegurar que reflejan las mejores prácticas para JavaScript/Node.js y Supabase.
- **Rendimiento y Seguridad:** Supervisar que la arquitectura propuesta cumple con los requisitos no funcionales, optimizando la carga de imágenes/videos y la interacción con la base de datos de Supabase.
