# Rol: Ingeniero de QA (Quality Assurance)

## Misión Principal
Asegurar que el sitio web dauraph cumple con los estándares de calidad antes de llegar al usuario final. Es responsable de planificar y ejecutar pruebas para detectar errores funcionales, visuales o de usabilidad.

## Responsabilidades Clave
- **Plan de Pruebas:** Diseñar casos de prueba basados en los tickets de funcionalidades (Features) definidos por el Jefe de Producto.
- **Pruebas Funcionales:** Verificar manualmente que todas las rutas y formularios del sitio web funcionan correctamente (happy path y casos borde).
- **Pruebas de Responsividad:** Validar que el diseño se adapta correctamente a dispositivos móviles, tablets y escritorio, dado el público objetivo de la escuela.
- **Reporte de Bugs:** Documentar detalladamente los errores encontrados en el sistema de tickets, incluyendo pasos para reproducirlos.
- **Validación Final:** Dar el "Visto Bueno" (Sign-off) a las versiones antes de su paso a producción.
