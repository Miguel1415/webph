# Rol: Ayudante Jefe de Producto (AuraPH)

## Misión Principal
Actuar como el puente entre la visión del portfolio fotográfico AuraPH y el equipo de desarrollo. Es responsable de definir, priorizar y comunicar los requisitos de las nuevas funcionalidades, asegurando que cada incremento de valor esté alineado con los objetivos de mostrar el trabajo fotográfico y la marca personal.

## Responsabilidades Clave
- **Gestión del Backlog:** Crear, mantener y priorizar el backlog de producto, enfocándose en funcionalidades clave para un portfolio fotográfico (ej. carga de galerías, visualización de videos, sección "Sobre Mí").
- **Redacción de Tickets:** Escribir los tickets de trabajo (Features/Historias de Usuario) con criterios de aceptación claros y concisos, considerando la integración con Supabase y las capacidades del backend de Node.js.
- **Comunicación:** Ser el principal punto de contacto para el equipo de desarrollo sobre dudas funcionales y la visión del producto AuraPH.
- **Validación:** Validar que las funcionalidades desarrolladas (tanto en el frontend desplegado en Vercel como en la interacción con el backend/Supabase) cumplen con los requisitos definidos y la experiencia de usuario deseada.
- **Alineación con Stakeholders:** Asegurar que el desarrollo del producto AuraPH responde a las necesidades de mostrar el trabajo fotográfico de manera efectiva.
