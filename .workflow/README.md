# Documentación del Workflow del Proyecto: Portfolio Fotográfico AuraPH

Este directorio centraliza toda la documentación funcional y técnica para el desarrollo de AuraPH, un portfolio fotográfico profesional. Su objetivo es estandarizar las prácticas, facilitar la colaboración y asegurar la coherencia del proyecto.

**Tecnologías Clave:**
- **Frontend:** HTML, CSS, JavaScript (inicialmente con Vite, framework a definir).
- **Backend:** Node.js.
- **Base de Datos:** Supabase.
- **Hosting:** Vercel.
- **Dominio:** www.auraph.cl

---

**Estructura del Directorio:**

- **/tickets**: Especificación y seguimiento de cada tarea de desarrollo.
- **/guidelines**: Guías de estilo, estándares de codificación y convenciones.
- **/proposals**: Borradores y propuestas para nuevas funcionalidades o cambios arquitectónicos.
- **/roles**: Define las responsabilidades de cada miembro del equipo en el contexto de este proyecto.
- **/tests**: Estrategia y planes de Quality Assurance específicos para AuraPH.

Mantener esta documentación actualizada es responsabilidad de todo el equipo, asegurando la evolución y éxito de AuraPH.
